<h1>Aplicación de pruebas con React Native</h1>

<h3>Instalación</h3>

<ul>
  <li>Instalar las dependencias con npm
    <code>
      npm install
    </code>
  </li>
  
  ---

  <li>
    Instalar paquetes de Expo
    <code>
      expo install
    </code>
  </li>

  ---

  <li>
    Correr aplicación
    <code>
      npm run start
    </code>
  </li>
</ul>
